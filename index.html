Securely send & receive money — student-friendly demo

Welcome to FarPay

Send and receive digital money, keep a wallet balance, get transaction emails and use 2FA for extra security. This demo is built with real security patterns — but uses dummy payments.

Quick features

  • Signup & secure login
  • 2FA setup (TOTP)
  • Wallet & add funds (dummy)
  • Send by email/username
  • Transaction history + email alerts
Built for learning — uses internal CSS & JS in each page. Use Node.js to run the server.
